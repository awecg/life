## 基本概念
+ 股息率: 每股分红 / 股价  一般能有3%就很不错了
+ 分红率：分红总额 / 总利润
+ 每股收益(earing per share): 税后利润 / 股本总数
+ 市盈率
  + TTM——Trailing Twelve Months 股价 / 最近四个季度的EPS
  + 静态 LYR——Last Year Ratio   股价 / 上一年度的EPS
